Eine kleine Anwendung, um .exe- und .msi-Dateien während des Signierens mit dem Versionsstring im Namen zu versehen.

Durch Aufruf mit dem Parameter 'params' können die an signtool.exe zu übergebenden Parameter im lokalen Profil des Benutzers gespeichert werden.

A small application to add a version string to the filename while signing .exe and .msi files.

The parameters signtool.exe should be called with can be saved to the user's lcal profile by calling the app with 'params' parameter.
